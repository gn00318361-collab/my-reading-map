# 🌟 奇遇的思維宇宙 (My Reading Map)

這是一個個人的數位花園與知識管理系統，將閱讀筆記、生活感悟透過「心智圖 (Mind Map)」與「部落格 (Blog)」的形式呈現。

🔗 **線上閱讀 (Live Site)**: [https://gn00318361-collab.github.io/my-reading-map/](https://gn00318361-collab.github.io/my-reading-map/)

## 🧠 核心設計 (Core Design)

### 1. 視覺化導航 (Markmap Visualization)
首頁 (`index.html`) 使用 **Markmap** 技術，將文章以思維導圖的方式展開。不同於傳統的列表，這種方式更能展現知識點之間的**關聯性**。

### 2. 卡片盒筆記法 (Zettelkasten Style)
節點顏色代表該概念在思考中出現的頻率與重要性：
*   **🔴 紅色 (高頻)**：出現 3 次以上的核心概念（如：大腦認知、職人精神）。
*   **🟡 黃色 (發展中)**：正在萌芽的想法（如：故事的力量）。
*   **🟢 綠色 (新知)**：單次出現的觀點。
*   **🟣 紫色 (跨書對話)**：不同書籍之間的概念共鳴與連結。

### 3. 整合式閱讀體驗
點擊心智圖節點，會直接跳轉至 `blog.html` 中的對應文章內容，提供無縫的閱讀體驗。收錄了 50+ 篇關於心理學、教育、財富與人性的深度筆記。
